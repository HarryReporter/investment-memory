# 投资决策框架

结构化的投资决策工具，帮助评估投资机会的风险和收益。

## 核心工具

- [[估值检查]] - 检查资产估值是否合理
- [[仓位管理]] - 根据风险等级计算建议仓位
- [[风险评估]] - 评估投资机会的综合风险

## 快速参考

### 估值检查清单
- PE/PB历史分位数：0-20%低估，20-50%合理偏低，50-80%合理偏高，80-100%高估
- PEG < 1可能低估，1-2合理，>2可能高估

### 仓位管理原则
- 凯利公式：仓位 = (胜率 × 赔率 - 败率) / 赔率
- 风险等级：极高5%，高5-10%，中10-20%，低20-30%
- 单股≤20%，单行业≤30%，保持10-20%现金

### 风险评估维度
- 市场风险25%，公司风险25%，流动性风险20%，政策风险15%，技术风险15%
