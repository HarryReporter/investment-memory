# 仓位管理

根据风险等级计算建议仓位的系统方法。

## 凯利公式简化版

```
仓位比例 = (胜率 × 赔率 - 败率) / 赔率
```

## 风险等级对应仓位

> [!info] 仓位比例
> - **极高风险**: 5%以内
> - **高风险**: 5-10%
> - **中等风险**: 10-20%
> - **低风险**: 20-30%

## 分散化原则

- 单只股票仓位不超过20%
- 单个行业仓位不超过30%
- 保持10-20%现金

```bash
uv run python skills/investment-framework/scripts/position_sizing.py \

  --win-rate 0.6 \
  --risk-reward 2 \
  --total-capital 100000
```
