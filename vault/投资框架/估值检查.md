# 估值检查

检查资产估值是否合理的系统方法。

## PE/PB历史分位数

> [!info] 估值区间
> - **0-20%**: 低估区域，可考虑加仓
> - **20-50%**: 合理偏低，可持有
> - **50-80%**: 合理偏高，谨慎
> - **80-100%**: 高估区域，考虑减仓

## 同行业对比

- 相对估值是否偏离行业平均
- 增长率是否匹配估值水平

## 增长率匹配度（PEG）

> [!tip] PEG判断
> - **PEG < 1**: 可能低估
> - **PEG 1-2**: 合理估值
> - **PEG > 2**: 可能高估

## 使用方法

```bash
uv run python skills/investment-framework/scripts/check_valuation.py \
  --code "HK.00700" \
  --pe 15 \
  --industry-pe 20 \
  --growth-rate 25
```
